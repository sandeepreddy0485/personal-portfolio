# Personal Portfolio

This project is a React-based personal portfolio website built with Create React App and deployed to GitHub Pages.

## Live Demo

View the site here: https://sandeepreddy0485.github.io/personal-portfolio/

## Features

- Responsive one-page portfolio layout
- About, skills, experience, certificates, projects, achievements, and contact sections
- Downloadable resume and certificate links
- GitHub Pages deployment setup

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode at http://localhost:3000.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the app for production into the `build` folder.

### `npm run deploy`

Builds the app and publishes it to GitHub Pages.

## Deployment

This project is configured for GitHub Pages using the `homepage` field in `package.json` and the `gh-pages` package.

To deploy updates:

```bash
npm run deploy
```

## Project Structure

- `src/` — React components and styling
- `public/` — Static assets such as the resume and certificates
- `build/` — Production build output generated by the deployment process
